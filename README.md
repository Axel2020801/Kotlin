# API 3: Consumo de API en Kotlin

## Descripción del Proyecto
Módulo desarrollado en Kotlin para el consumo de una API REST de E-Commerce, modelado de datos mediante Data Classes y procesamiento de respuestas en formato JSON. Diseñado como base para pruebas automatizadas de Mobile Testing.

## Estructura del Proyecto
* `Product.kt`: Modelo de datos para los productos de la tienda.
* `ProductRepository.kt`: Gestor de peticiones de red y mapeo de datos JSON.
* `Main.kt`: Punto de entrada de la aplicación y simulación de consola.

## Tecnologías Utilizadas
* Kotlin JVM
* JSON para el procesamiento de datos de la API.
